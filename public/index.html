POTrack connects to Xero via OAuth — your credentials stay secure on the server.

1
Create a Xero developer app
Go to developer.xero.com → My Apps → New App. Choose Web app.
2
Set the redirect URI
Add: https://your-app-url/auth/xero/callback
3
Add credentials to your server
Set XERO_CLIENT_ID and XERO_CLIENT_SECRET in your Railway environment variables.
4
Connect below
Connect to Xero

This opens Xero's login page. Approve access and you'll be connected automatically. Credentials are stored securely on the server, never in the browser.
